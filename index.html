Comprehensive Management System

The Comprehensive Management System is an all-in-one platform that integrates multiple business functions—POS, Hotel Booking, Employee Management, Payroll, Inventory, and Accounting—into one seamless web-based solution.

Key Features
  • POS with batch-level inventory, reorder alerts, and sales receipt generation.
  • Hotel Booking module with room availability, duration-based pricing, and booking management.
  • Employee module with QR-based attendance and automated scheduling.
  • Payroll processing with holiday pay, overtime, and leave deductions.
  • Accounting integration for journal entries, invoices, and balance tracking.
  • Dynamic dashboard summarizing sales, low stock, payroll, and bookings.
Technologies Used

HTML, CSS, Bootstrap, JavaScript (AJAX), PHP (MVC with PDO), and MySQL.
